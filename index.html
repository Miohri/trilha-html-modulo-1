História dos computadores

Primeiros computadores

1a geração

  • Harvard Mark I (1944): Grande máquina de cálculo eletro mecânica.
  • Colossus (Alemanha - 1946): Cálculo e criptografia.
  • ENIAC (1946): Mais automático que o Mark I.

2a geração (1959 a 1965)

Transistores começaram a diminuir o tamanho dos computadores.


3a geração (1965 a 1970)

Circuitos integrados (chips). Maior diminuição.


4a geração (a partir de 1971)

Surgimento do personal computer.

O que são Clients?

Cliente no Linguajar Web:

Clientes consomem a informação de algum lugar, ou seja, um cliente é o dispositivo que está consumindo aquela informação na internet.

Os navegadores juntam cache de site para que não haja necessidade de buscar e baixar novamente no servidor. - Ao utualizar a página (F5) o cache é apagado e as informações são buscadas novamente.

Diferentes resolusões e compatibilidade entre navegadores devem ser consideradas na criação de websites.

O que são Servers?

Sobre Servidores:

São computadores ou sistemas projetados para fornecer serviços ou recursos aos clientes em uma rede. Existem diversos tipos: de arquivos, segurança (firewall), streaming, e-mail e web.

Linguagens de Programação

O que é?

Linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).


Server-Side:

São linguagens de programação que rodam do lado do servidor. Por exemplo: PHP, Ruby, C#, JavaScript (V8, Node), Java.


Client-Side:

Linguagens que são executadas no cliente. Ex: JavaScript.


HTML não é Linguagem de Programação (na verdade, é um arquivo de marcação.)
